Overview


This week I've been working on all sorts of different projects. First i've decived to touch up on Reflex, completely remastering it. I've completely changed the game mode, art style and got rid of the old code created by 13 year old me. So be sure to check that out!

Operation Blood Count

Now for the new project. Operation Blood Count was the project I originally started this summer with plans to finish before I went off to college. However towards the end of July I was satisfied with the direction the game was heading. Taking a break from it, while starting up Exhiled, I took time to rethink where I wanted the game to go and what I really wanted to achieve with it. Now that I have figured out the direction of the game, i'm getting back to work on it. You can download the original Legacy version of the game from the 'Our Games' Section, or wait for a future update on it here in the Dev Blogs. I will continue working on Exhiled as well, but this game will also be within my focus.



Exhiled
