Surviving Exile




Current Project In Development

Man oh man, has this project had to jump through some hoops. Originally, I had started this project as a desire to create a new challenge for myself, while taking a break from mobile games. Exhiled formed with the attempt to create everything from scratch. Rather then my typical procedure of dragging around art assets, I wanted to create an entire world from code, from procedural code, to inventories and complex UI structures, thus the idea for Exhiled was formed.

While originally this project was initially due for an initial pre-alpha release towards the end of November, in 2017, I decided to rework a lot of the game, completely remodeling the art style from medieval to more of an abstract space theme. With that art style change came a lot of large game mechanic reworks, thus delaying the initial alpha release of the entire project. Further more, the project was continously delayed due to large amounts of school work piling up, leading to over 6 months of no additional work on the project.

As of June 2018, the Exhiled Project is currently aimmed at being a colony management game with a twist. Hopes of an inital pre-alpha release are targeted for early August. Keep up with the dev blogs for any new information regarding this release.
